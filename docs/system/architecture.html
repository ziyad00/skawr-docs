🔐 Authentication Flow

User registration, API key generation, session management, and authentication validation flows with bcrypt security.

sequenceDiagram participant U as User participant D as Dashboard participant API as FastAPI Backend participant DB as PostgreSQL participant SM as Secrets Manager Note over U,SM: User Registration & API Key Generation U->>D: Sign up with email/password D->>API: POST /api/v1/public/signup API->>DB: Create client record API->>DB: Generate & store API key (bcrypt hash) API-->>D: Return API key D->>D: Store API key in localStorage D-->>U: Show dashboard Note over U,SM: Subsequent API Requests U->>D: Navigate to dashboard page D->>D: Get API key from localStorage D->>API: GET /api/v1/saas/indices
X-API-Key: sk-xxxx API->>API: Extract API key prefix API->>DB: Query keys by prefix API->>API: bcrypt.verify(provided_key, stored_hash) alt Valid API Key API->>DB: Update last_used_at API->>API: Process request API-->>D: Return data D-->>U: Show content else Invalid API Key API-->>D: 401 Unauthorized D->>D: Clear localStorage D-->>U: Redirect to login end Note over U,SM: Session Management U->>D: Refresh page D->>D: Check localStorage for API key D->>API: GET /api/v1/saas/usage (validate key) alt Key Valid API-->>D: 200 OK D-->>U: Stay logged in else Key Invalid/Expired API-->>D: 401 Unauthorized D->>D: Clear localStorage & redirect D-->>U: Show login page end

📋 Architecture Summary

🏗️ Infrastructure

  • AWS ECS Fargate for container orchestration
  • Application Load Balancer for traffic distribution
  • PostgreSQL RDS for structured data
  • OpenSearch for full-text and vector search
  • S3 for file storage and backups
  • CloudFront CDN for static asset delivery

🔧 Application Services

  • FastAPI backend with Python
  • React frontend with TypeScript
  • Multi-tenant SaaS architecture
  • API key authentication with bcrypt
  • Rate limiting per client/API key
  • Real-time search and analytics

🚀 DevOps & Deployment

  • GitHub source control with protected branches
  • CodePipeline for CI/CD automation
  • Automated database migrations
  • Rolling deployments with health checks
  • CloudWatch monitoring and alerting
  • Automatic rollback on deployment failure

💾 Data Architecture

  • Multi-tenant database design
  • Hybrid search (keyword + vector embeddings)
  • Real-time data synchronization
  • Usage analytics and tracking
  • Bulk data import/export capabilities
  • Data validation and transformation pipeline
